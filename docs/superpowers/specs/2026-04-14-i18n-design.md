# Polyant 多语言支持设计文档

## 概述

为 Polyant 添加多语言支持，初始支持中文（zh-CN）和英文（en-US），架构设计支持未来扩展更多语言。

### 需求范围

| 模块 | 多语言支持内容 |
|------|---------------|
| API 响应 | 成功/错误消息 |
| CLI 输出 | 命令输出、帮助信息 |
| 服务日志 | 运行时日志消息 |
| Web 宣传页 | 页面内容切换 |
| 数据模型 | 条目语言属性、分类多语言名称 |

// pkg/i18n/i18n.go
package i18n

type Lang string

const (
    LangZhCN Lang = "zh-CN"
    LangEnUS Lang = "en-US"
)

// Translator 翻译器接口
type Translator interface {
    T(code string, args ...interface{}) string
    Tc(ctx context.Context, code string, args ...interface{}) string
    TWithLang(lang Lang, code string, args map[string]interface{}) string
}

// 全局函数
func T(code string, args ...interface{}) string
func Tc(ctx context.Context, code string, args ...interface{}) string
func GetLangFromRequest(r *http.Request) Lang

### API 响应处理

```go
// internal/api/handler/helpers.go
func writeError(w http.ResponseWriter, r *http.Request, err *awerrors.AWError) {
    lang := i18n.GetLangFromRequest(r)
    message := i18n.TWithLang(lang, err.I18nCode)
    if message == "" {
        message = err.Message // 回退到默认消息
    }
    
    writeJSON(w, err.HTTPStatus, &APIResponse{
        Code:    err.Code,
        Message: message,
    })
}
```

## 语言检测流程

### 优先级顺序

1. **请求参数**: `?lang=zh-CN`
2. **HTTP Header**: `Accept-Language: zh-CN,zh;q=0.9,en;q=0.8`
3. **用户偏好**: 用户注册时设置的语言偏好
4. **全局配置**: 配置文件中的 `default_lang`

### 实现代码

```go
// pkg/i18n/context.go
func GetLangFromRequest(r *http.Request) Lang {
    // 1. URL 参数
    if lang := r.URL.Query().Get("lang"); lang != "" {
        return Lang(lang)
    }
    
    // 2. Accept-Language Header
    if acceptLang := r.Header.Get("Accept-Language"); acceptLang != "" {
        return parseAcceptLanguage(acceptLang)
    }
    
    // 3. 用户上下文（已登录用户）
    if user := middleware.GetUserFromContext(r.Context()); user != nil {
        if user.PreferredLang != "" {
            return Lang(user.PreferredLang)
        }
    }
    
    // 4. 全局默认
    return globalDefaultLang
}
```

## 日志系统改造

### Logger 扩展

```go
// pkg/logger/logger.go
type Logger struct {
    level      int
    lang       i18n.Lang
    bilingual  bool
    // ... 其他字段
}

func (l *Logger) Info(code string, args map[string]interface{}) {
    l.log(LevelInfo, code, args)
}

func (l *Logger) log(level int, code string, args map[string]interface{}) {
    levelName := levelNames[level]
    
    if l.bilingual {
        msgZh := i18n.TWithLang(i18n.LangZhCN, code, args)
        msgEn := i18n.TWithLang(i18n.LangEnUS, code, args)
        l.logger.Printf("[%s] %s | %s", levelName, msgZh, msgEn)
    } else {
        msg := i18n.TWithLang(l.lang, code, args)
        l.logger.Printf("[%s] %s", levelName, msg)
    }
}
```

### 使用示例

```go
// 旧方式
logger.Info("服务启动完成，监听端口 %d", 8080)

// 新方式
logger.Info("log.server.started", map[string]interface{}{"port": 8080})
```
